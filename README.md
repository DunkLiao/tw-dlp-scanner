# DLP 敏感資料掃描工具

DLP 敏感資料掃描工具是一個使用 Python `tkinter` 與 `ttk` 製作的桌面應用程式，可讓使用者選擇指定資料夾，掃描資料夾內檔案是否包含個資、敏感資料、加密檔案或特定機密關鍵字。

本工具適合用於內部資料夾初步盤點、稽核前檢查、敏感資料外洩風險檢視，以及文件歸檔前的快速掃描。

---

## 功能特色

- 圖形化介面，使用 Python 內建 `tkinter.ttk`
- 可選擇指定資料夾掃描
- 可選擇是否包含子資料夾
- 掃描過程使用背景執行緒，避免 GUI 卡住
- 支援掃描多種常見檔案格式
- 支援掃描 ZIP 內部檔案與巢狀 ZIP
- 支援信用卡號 Luhn 檢核
- 支援加密檔案偵測
- 掃描結果可直接匯出 Excel 報表
- Excel 報表包含風險等級與命中摘要
- 命中內容會自動遮罩部分敏感資訊
- 掃描完成會跳出提示訊息
- 內含 `build_exe.bat`，可一鍵打包成 Windows EXE

## 支援檔案格式

| 類型 | 副檔名 |
|---|---|
| 純文字 | `.txt`, `.csv`, `.log`, `.md`, `.json`, `.xml`, `.html`, `.htm` |
| Word | `.doc`, `.docx` |
| Excel | `.xls`, `.xlsx` |
| PowerPoint | `.ppt` |
| PDF | `.pdf` |
| ZIP 內部掃描 | `.zip`，支援遞迴掃描 ZIP 內的文字、Office、PDF 與巢狀 ZIP |
| 加密檔案偵測 | `.docx`, `.xlsx`, `.pptx`, `.pdf`, `.zip`, `.rar`, `.7z` |

> 注意：`.doc` 與 `.ppt` 採 OLE 文字最佳努力抽取；若內容是圖片、特殊二進位結構或複雜版面，可能無法完整擷取。`.rar`, `.7z`, `.msg`, `.eml` 尚未做內文解析。

## 偵測項目

| 編號 | 類型 | 說明 |
|---|---|---|
| 1 | 組合式個資 | 同一檔案中命中兩種以上個資類型 |
| 2 | 加密檔案 | 偵測 Office、PDF、ZIP 可能加密的檔案 |
| 3 | 台灣地址 | 偵測台灣縣市、路、街、巷、弄、號等地址格式 |
| 4 | 中文姓名 | 搭配稱謂或職稱判斷中文姓名 |
| 5 | 台灣身分證字號 | 偵測台灣身分證格式 |
| 6 | 護照號碼 | 偵測 9 碼數字格式 |
| 7 | 信用卡號 | 偵測 13 至 19 碼數字，並使用 Luhn 演算法檢核 |
| 8 | 銀行存摺根號與關鍵字 | 偵測「客戶往來明細查詢」與帳號格式 |
| 9 | 公文密等關鍵字組合 | 偵測發文字號、密等及解密條件、保密期限等關鍵字組合 |
| 10 | 電子郵件 | 偵測 Email 格式 |
| 11 | 居留證號碼 | 偵測新版與舊版居留證格式 |

## 風險等級

| 風險等級 | 類型範例 |
|---|---|
| 高 | 身分證字號、信用卡號、護照號碼、居留證號碼、公文密等、銀行存摺根號 |
| 中 | 台灣地址、手機號碼、電話號碼、加密檔案 |
| 低 | 中文姓名、電子郵件 |

Excel 報表會依風險等級自動標色：

| 風險 | 顏色 |
|---|---|
| 高 | 淺紅色 |
| 中 | 淺黃色 |
| 低 | 淺綠色 |

## 使用方式

1. 點選「瀏覽」
2. 選擇要掃描的資料夾
3. 勾選是否要「包含子資料夾」
4. 點選「開始掃描」
5. 等待掃描完成
6. 若有命中，畫面下方會列出：
   - 風險等級
   - 檔案名稱
   - 命中類型
   - 命中次數
   - 命中範例
   - 完整路徑
7. 點選「匯出 Excel」可產出掃描報表

## 信用卡 Luhn 檢核說明

本工具針對信用卡號不只使用格式比對，也加入 Luhn 檢核。

流程如下：

1. 先找出疑似 13 至 19 碼的數字組合
2. 移除空白與連字號
3. 使用 Luhn 演算法檢核
4. 通過檢核才判定為信用卡號命中

這可以降低一般帳號、流水號、單號被誤判成信用卡號的機率。

---

## 加密檔案偵測說明

| 檔案 | 偵測方式 |
|---|---|
| PDF | 嘗試開啟 PDF，若發生 password 或 encrypted 相關錯誤則判定可能加密 |
| DOCX/XLSX/PPTX | 嘗試用 ZIP 結構開啟，若不是一般 OOXML 結構則可能為加密 Office 檔 |
| ZIP | 檢查 ZIP entry 的 encryption flag |
| RAR/7Z | 目前僅列入副檔名範圍，尚未做深度解析 |

## 打包成 Windows EXE

若要提供給沒有安裝 Python 的使用者使用，可以透過 `build_exe.bat` 一鍵打包成 Windows 執行檔。

### 打包方式

在 Windows 環境中，直接雙擊：

```text
build_exe.bat
```

批次檔會自動執行以下動作：

1. 檢查 Python 是否可用
2. 安裝 `requirements.txt` 內的必要套件
3. 安裝 PyInstaller
4. 清除舊的 `build` 與 `dist` 資料夾
5. 重新打包成單一 EXE 檔案

### 打包完成位置

```text
dist/DLPScanner.exe
```

可將 `DLPScanner.exe` 複製給其他 Windows 使用者執行。

## 隱私與安全提醒

本工具會在本機執行，不會主動上傳檔案到外部系統。

仍需注意：

- 掃描結果可能含有敏感資訊
- 匯出的 Excel 報表應妥善保存
- 建議將報表放在受控權限的資料夾
- 不建議將掃描報表寄送到外部信箱
- 命中內容雖然有遮罩，但仍可能保留部分關鍵資訊

---

## 已知限制

1. `.doc` 與 `.ppt` 使用 OLE 文字最佳努力抽取，準確率低於先轉成 `.docx/.pptx` 後掃描
2. 掃描 PDF 時，若 PDF 是圖片型掃描檔，可能無法擷取文字
3. 中文姓名偵測依賴稱謂或職稱，可能漏判
4. 地址格式變化很多，可能會有誤判或漏判
5. 護照號碼目前以 9 碼數字判斷，可能與其他流水號混淆
6. RAR 與 7Z 尚未做壓縮檔內文解析
7. ZIP 內部掃描有深度與大小限制，避免大型或惡意壓縮檔造成掃描卡住
8. OCR 文字辨識尚未支援
